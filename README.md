# Tesla Financial Analysis 

End-to-End Data Analysis Project using Python, SQL, and Power BI.

## Project Overview
This project analyzes Tesla's financial performance from 2019–2025.

The workflow includes:
- Data creation using Python
- Data storage and analysis using SQL
- Data visualization using Power BI

---

## Tools Used
- Python (Pandas)
- SQL Server
- Power BI
- GitHub

---

## Key Analysis
- Revenue Growth Analysis
- Net Income Trend
- Automotive vs Energy Revenue Comparison
- Profit Margin Calculation
- Revenue Breakdown by Segment

---

## Files Included
- Tesla_Data.py → Dataset creation script
- SQL_Analysis.sql → SQL queries for analysis
- tesla_yearly_data.csv → Dataset
- Tesla_Financial_Dashboard.pbix → Power BI Dashboard
- dashboard_preview.png → Dashboard Preview
